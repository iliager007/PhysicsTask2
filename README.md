# 3-D Симуляция ткани — PBD / XPBD, Projective Dynamics, VBD

Браузерная **3-D** симуляция ткани, полностью написанная на Python с использованием **Pyodide** (Python + NumPy, скомпилированные в WebAssembly).

## Солверы

| Солвер | Описание |
|---|---|
| **PBD / XPBD** | Расширенная позиционная динамика (Müller и др.). Итеративная проекция ограничений с compliance-параметром |
| **Projective Dynamics** | Метод Bouaziz и др. (SIGGRAPH 2014). Локально-глобальная оптимизация с предварительно факторизованной системной матрицей |
| **VBD (Vertex Block Descent)** | Метод Chen и др. (SIGGRAPH 2024). Покоординатный спуск с решением локальной 3×3-системы Ньютона для каждой вершины |

## Управление

| Ввод | Действие |
|---|---|
| **Левая кнопка мыши** на холсте | Захват и перемещение ближайшей частицы |
| **Правая кнопка мыши** на холсте | Вращение камеры (орбита) |
| **Колесо мыши** | Приближение / отдаление |
| **Ползунки** | Итерации, жёсткость, демпфирование, упругость отскока, трение пола |

## Сцены

### 1 · Подвес на фиксированных точках

| № | Сцена | Описание |
|---|---|---|
| 0 | Одна центральная точка | Ткань висит на одном закреплении — 3D-маятник |
| 1 | Два верхних угла | Классическая драпировка «занавес» с препятствием-шаром |
| 2 | Три точки | Левая / центральная / правая точки закрепления |
| 3 | Весь верхний край | Вся верхняя строка закреплена |

### 2 · Столкновения с полом и шаром

| № | Сцена | Описание |
|---|---|---|
| 4 | Свободное падение на пол | Без закреплений — ткань падает, отскакивает, скользит с трением |
| 5 | Падение на шар | Ткань обволакивает шар, затем сползает на пол |
| 6 | Плотное сминание | Мелкая сетка сминается — выражены самоколлизии |

### 3 · Переопределённые / несовместимые ограничения

| № | Сцена | Описание |
|---|---|---|
| 7 | Два удалённых угла | Углы разнесены на 1.8× от естественной ширины |
| 8 | Четыре угла растянуты | Все углы разнесены на 1.7× от естественного размера |
| 9 | Сжатые края | Края сжаты до 35 % — ткань выпучивается |

## Локальный запуск

```bash
python -m http.server 8000   # или: npx serve .
```

Откройте `http://localhost:8000`.
